# WeatherNow

A feature-rich Flutter weather app with real-time data, Firebase Firestore integration, and a polished dark UI.

## Features

- Real-time weather by city name (OpenWeatherMap API)
- 24-hour hourly forecast with icons
- Celsius / Fahrenheit toggle
- Humidity, wind speed + direction, visibility, pressure
- Sunrise and sunset times with visual arc
- Dynamic background gradient based on weather condition and time of day
- Firebase Firestore — search history saved automatically
- History screen — view, tap to re-search, swipe to delete, clear all
- Firebase status indicator (green dot = connected, red = error)
- Pull-to-refresh
- Recent searches as quick-access chips

## Tech Stack

- Flutter 3.x
- OpenWeatherMap API (free tier)
- Firebase Core + Cloud Firestore

## Setup

### 1. OpenWeatherMap API key

Sign up free at [openweathermap.org](https://openweathermap.org). Go to **My API Keys**, copy the default key, paste it in `lib/services/weather_service.dart`:

```dart
static const String _apiKey = 'your_key_here';
```

### 2. Firebase

1. Create a project at [console.firebase.google.com](https://console.firebase.google.com)
2. Enable **Firestore Database** in Test mode
3. Run:

```bash
dart pub global activate flutterfire_cli
npm install -g firebase-tools
firebase login
flutterfire configure
```

### 3. Run

```bash
flutter pub get
flutter run -d chrome
```

## Confirming Firebase is connected

- A green dot labeled **Firebase connected** appears below the top bar when Firestore is reachable
- A red warning appears if the connection fails
- Open the **History** screen (clock icon) to see all saved searches pulled live from Firestore
